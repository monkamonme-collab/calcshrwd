# CalcShrwd - 免费在线计算器工具集

一个基于 Next.js 的免费在线计算器集合，提供多种实用计算工具。

## 包含计算器

- **BMI 计算器** (`/bmi-calculator`) — 身体质量指数计算
- **GPA 计算器** (`/gpa-calculator`) — 学业平均绩点计算
- **小费计算器** (`/tip-calculator`) — 餐厅小费与人均分摊
- **贷款计算器** (`/loan-calculator`) — 月供与还款总额计算
- **薪资计算器** (`/paycheck-calculator`) — 税后薪资估算
- **百分比计算器** (`/percentage-calculator`) — 百分比增减与比值计算
- **字数计算器** (`/word-counter`) — 字数、字符数与阅读时间统计
- **复利计算器** (`/compound-interest-calculator`) — 复利增长与投资回报估算
- **成绩换算器** (`/grade-calculator`) — 分数与字母等级互换

## 本地开发

```bash
npm install
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看结果。

## 部署

生产环境部署于：**https://calcshrwd.vercel.app**

## 技术栈

- [Next.js](https://nextjs.org/) — React 全栈框架
- [TypeScript](https://www.typescriptlang.org/) — 类型安全
- [Tailwind CSS](https://tailwindcss.com/) — 原子化 CSS 样式
- [Vercel](https://vercel.com/) — 托管与持续部署
